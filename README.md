# BilibiliTranscoding
Biliblli视频转码,无需登录即可观看高清画质,支持MP4格式下载本地，无去水印功能,自测适用于大部分视频，网上还有更方便的图形化工具，Just For Fun  仅供娱乐。

首先打开网页B站，选择视频，复制视频链接，运行`python3 BilibiliTranscodeing.py -u http://www.bilibili.com/video/BVXXXXX`
